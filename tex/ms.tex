\begin{abstract}
The observed-frame optical spectrum of each extragalactic DESI target
contains a wealth of information beyond its redshift. \fastspecfit{}
(\url{https://fastspecfit.readthedocs.io}) is open-source Python software
that extracts this information by jointly modeling each DESI spectrum and
its broadband photometry using physically motivated stellar continuum and
emission-line templates. The code operates in two modes: \fastphot{} fits
broadband spectral energy distributions alone, while \fastspec{} performs
a simultaneous spectrophotometric fit. Measured quantities include
light-weighted stellar ages, velocity dispersions, stellar masses,
star-formation rates, K-corrections, rest-frame absolute magnitudes and
colors, and integrated fluxes and equivalent widths for more than 40
emission lines spanning the rest-frame UV through near-infrared. We
describe the algorithms, validate the measurements using repeat
observations and comparisons to complementary codes, and present the
\fastspecfit{} value-added catalog for the DESI Data Release 2,
comprising \todo{N} million extragalactic targets.
\end{abstract}

cut. Either way, all photometry is ultimately assigned \texttt{PHOTSYS}
$=$ \texttt{N} or \texttt{S}, which uniquely associates the photometry
with the appropriate filter curves.

\subsection{SPS Template Library}
\label{ssec:templates}

\begin{itemize}
  \item FSPS with C3K stellar library (2750--9100~\AA, $R = 3000$), MIST
    isochrones \citep{choi16a, dotter16a}, Chabrier IMF \citep{chabrier03a}
  \item Non-parametric SFH: 5 lookback-time bins (30~Myr to 13.7~Gyr)
  \item Solar metallicity default; nebular emission included via FSPS grid
  \item Auxiliary templates: Draine \& Li (2007) dust emission, Nenkova et
    al.\ (2008) AGN torus, Vestergaard \& Wilkes (2001) Fe emission
  \item Wavelength grid: 25~\kms\ pixels in 2750--9100~\AA; native FSPS
    grid outside this range
\end{itemize}

For the DESI galaxy templates we use the Flexible Stellar Population
Synthesis (\texttt{FSPS}) modeling
code\footnote{\url{https://github.com/cconroy20/fsps}}
\citep{conroy09a, conroy10b} via the
\texttt{python-fsps}\footnote{\url{https://github.com/dfm/python-fsps}}
Python interface. The code includes the option of excluding templates
older than the age of the universe; however, by default we use all
templates, largely because of the relatively coarse grid in mean stellar
age \citep{leja17a, leja19a}.

\subsection{Dust Attenuation and Emission}
\label{ssec:dust}

\begin{itemize}

    $k(\lambda) \propto (\lambda / 5500\,\text{\AA})^{-0.7}$
  \item Single free parameter $\tauv \in [0, 2]$
  \item Energy-balance IR emission: absorbed luminosity reprocessed through

\end{itemize}

\subsection{Redshift, IGM, and Flux Scaling}
\label{ssec:zfactors}

\begin{itemize}
  \item Inoue et al.\ (2014) IGM transmission (Ly$\alpha$ forest,
    Lyman-limit systems)
  \item Luminosity distance, flux normalization, mass normalization
    ($\mathcal{M}_0 = 10^{10}\,\mathrm{M_\odot}$)
  \item Pre-computed \texttt{zfactors} array cached at object
    initialization
\end{itemize}

\subsection{Velocity Dispersion}
\label{ssec:vdisp}

\begin{itemize}
  \item Gaussian convolution on $\log\lambda$ grid
    ($\sigma_\mathrm{pix} = \vdisp / 25\,\kms$)
  \item FFT-based convolution; pre-computed transform cache up to
    500~\kms
  \item Fitting window: 3800--6000~\AA\ rest-frame
    (Ca\,H\&K, $G$-band, Mg\,\textsc{b}, Fe)
  \item Two-stage fit: $\chi^2$ grid scan then trust-region refinement
  \item Fallback: nominal 250~\kms\ when wavelength coverage is
    insufficient
\end{itemize}

The nominal velocity dispersion of 250~\kms\ is close to the knee of
the velocity dispersion function at $z = 0$--$1.5$
\citep[e.g.,][]{bezanson11a}. This value is an overestimate for the
emission-line galaxies targeted by DESI, but the effect on the continuum
fit is small because those targets are observed primarily in the
rest-frame UV where stellar absorption features are less prominent.

\subsection{Fitting Algorithm}
\label{ssec:fitting}

\begin{itemize}
  \item Variable projection (VARPRO): outer Brent scalar search over
    $\tauv$, inner non-negative least squares (NNLS) over template
    coefficients
  \item Aperture correction: fiber-to-total flux ratio applied before full
    fit
  \item Full spectrophotometric fit: spectra + photometry stacked
    residuals, $\vdisp$ fixed, nebular emission included
  \item Trust-region reflective (TRF) optimizer when $\vdisp$ is a free
    parameter
\end{itemize}

The aperture correction may not be perfect for QSOs, but in general our
templates have enough flexibility to recover the shape of the broadband
SED reasonably well \citep{leja17a, leja19a, cappellari22a}.

\subsection{Smooth Continuum Correction}
\label{ssec:smooth}

\begin{itemize}
  \item Nonparametric correction for residual flux-calibration errors and
    template mismatch, computed after SED subtraction
  \item Sliding-window robust median + quadratic spline fit per camera
  \item Added to SED model before emission-line fitting
\end{itemize}

\subsection{Uncertainties}
\label{ssec:continuum_uncertainties}

\begin{itemize}
  \item Monte Carlo: $N = 50$ independent noise realizations of spectra
    and photometry
  \item Uncertainties reported for $\tauv$, $\vdisp$ (when fitted),
    $D_n(4000)$, luminosities, and continuum fluxes
\end{itemize}

\subsection{Forward Model}
\label{ssec:emline_model}

\begin{itemize}
  \item Intrinsic profile: log-Gaussian in observed wavelength
  \item DESI resolution matrix: pre-convolved representation with
    effective width
    $\sigma_\mathrm{eff} = \sqrt{\sigma^2 + \sigma_G^2}$
  \item Analytic Jacobian for efficient nonlinear least-squares
  \item Removes 5--30\% flux bias present in earlier bin-integration
    approach for unresolved lines
\end{itemize}

After subtracting the best-fitting stellar continuum model from the data,
we are left with a residual spectrum that may contain emission lines. We
adopt a physically motivated forward-modeling approach to fit these
residuals.

We model the emission-line spectrum as a sum of Gaussian line profiles,
each described by a model amplitude $A_{m}$ (in \fluxdensity), velocity
shift $\Delta v$ (in \kms), and velocity width $\sigma$ (in \kms). The
velocity shift, measured with respect to the Redrock redshift $z$,
determines the redshifted wavelength of each line,
\begin{equation}
  \lambda_z = \lambda_{0} \left(1 + z + \frac{\Delta v}{c}\right),
\end{equation}
where $\lambda_{0}$ is the vacuum rest wavelength
(Table~\ref{table:emlines}) and $c$ is the speed of light. Working in
base-10 logarithmic wavelength space (constant-velocity pixel space), we
construct each model emission line as
\begin{equation}
  \widetilde{G}(A_{m}, \Delta v, \sigma, \lambda^{\prime}) =
  A_{m} \exp\!\left[
    -\frac{(\log_{10}\lambda^{\prime} - \log_{10}\lambda_z)^2}
          {2\sigma_{\log\lambda}^2}
  \right],
\end{equation}
where $\sigma_{\log\lambda} = \sigma / \log_{e}(10) / c$ is the velocity
width in $\log_{10}(\mathrm{\AA})$ and $\lambda^{\prime}$ is the model
wavelength array before resampling onto the DESI pixel grid. The full
emission-line model is
\begin{equation}
  \widetilde{\mathcal{G}} = \sum_{i=1}^{N} \widetilde{G}_i,
\end{equation}
where $N$ is the number of lines within the observed spectral range.
After flux-conserving trapezoidal resampling onto the DESI pixel grid,
we multiply by the resolution matrix $\mathcal{R}$ to obtain the final
model,
\begin{equation}
  \mathcal{G}(\pmb{A_m}, \pmb{\Delta v}, \pmb{\sigma}, \lambda) =
  \mathcal{R}\,\widetilde{\mathcal{G}}(\lambda^{\prime}\rightarrow\lambda).
  \label{eq:gtilde}
\end{equation}
In practice the parameters $\pmb{A_{m}}$, $\pmb{\Delta v}$, and
$\pmb{\sigma}$ are not all independent; physical constraints on
amplitudes, velocity centers, and widths reduce the number of free
parameters and improve reliability for weak or blended lines. Multiplying
by $\mathcal{R}$ decreases each model line's amplitude by an amount that
depends on the line width and wavelength. The observed amplitude $A$
(peak of $\mathcal{G}_i$) is therefore recorded separately after the
final fit, as it is the appropriate quantity for assessing line
significance.

\subsection{Fitting Algorithm and Upper Limits}
\label{ssec:emline_fitting}

\begin{itemize}
  \item Nonlinear least-squares on continuum-subtracted residuals,
    using the trust-region reflective algorithm
    \citep[\texttt{scipy.optimize.least\_squares};][]{branch99a}
  \item Significance threshold: $A\,\sqrt{I_A} > 1.5$, where $I_A$
    is the inverse variance of the observed line amplitude
  \item Upper limits and equivalent widths from adjacent continuum flux
    densities measured from the SED model \citep{andrae10a}
\end{itemize}

Given the model $\mathcal{G}$, we minimize
\begin{equation}
  \chi^2 = \frac{1}{2} \sum_{j=1}^{n}
  I_{j}\bigl(F_{j} - \mathcal{G}_{j}\bigr)^2,
  \label{eq:chi2line}
\end{equation}
where $F_{j}$ is the observed spectrum (in \fluxdensity) and $I_{j}$ is
the inverse-variance spectrum (in \ivardensity).

To derive the Gaussian-integrated flux and its uncertainty we apply the
matched-filtering formalism \citep[e.g.,][]{lang20a}. Let
$P_{j} \equiv G_{j} / \sum_{j} G_{j}$ be the normalized,
resolution-convolved line profile evaluated over the $\pm3\sigma$ pixel
window. The statistical weight of each pixel (in \ivarflux) is
\begin{equation}
  w_{j} = \frac{P_{j}^{2}}{(\Delta\lambda)_{j}^{2}}\,I_{j},
\end{equation}
where $\Delta\lambda_{j}$ is the pixel width in \AA. The
maximum-likelihood Gaussian-integrated flux is
\begin{equation}
  F_{\mathrm{Gauss}} =
  \frac{\displaystyle\sum_{j} w_{j}\,\Delta\lambda_{j}\,G_{j} / P_{j}}
       {\displaystyle\sum_{j} w_{j}},
  \label{eq:gaussflux}
\end{equation}
with inverse variance $I_{F_\mathrm{Gauss}} = \sum_{j} w_{j}$. Both
quantities are corrected by $\mathrm{erf}(3/\sqrt{2}) = 0.9973$ to
account for flux outside the $\pm3\sigma$ window.

\subsection{Stellar Masses and Star-Formation Rates}
\label{ssec:mstar_sfr}

\begin{itemize}
  \item Stellar mass from NNLS coefficients and FSPS surviving mass
    fractions
  \item SFR from emission lines (\ha, \oii) and/or UV continuum
  \item $D_n(4000)$ from line-free best-fit model \citep{balogh99a}

    $\sim$0.3--0.4~dex higher than parametric estimates
    \citep{leja19a, leja20a}

\subsection{K-corrections and Rest-Frame Magnitudes}
\label{ssec:kcorr}

\begin{itemize}
  \item K-corrections by comparing synthesized observer- and rest-frame
    photometry from the best-fit SED \citep{hogg02a, blanton07a}
  \item Rest-frame absolute magnitudes from distance modulus +
    K-correction
  \item Monochromatic luminosities at 1450, 1500, 1700, 2800, 3000, and
    5100~\AA
\end{itemize}

\subsection{Spectral Indices and Equivalent Widths}
\label{ssec:indices}

\begin{itemize}
  \item EWs computed relative to local continuum flux densities from the
    SED model
  \item $D_n(4000)$ measured from both the best-fit model and the observed
    spectrum \citep{balogh99a}
\end{itemize}
